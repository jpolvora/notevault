# Task Completion Checklist

Before finishing a task or submitting a PR, ensure:
1. [ ] Code follows the established patterns in the codebase.
2. [ ] `npm run lint` passes without errors.
3. [ ] `npm run typecheck` passes without errors.
4. [ ] `npm run format` is executed.
5. [ ] (If applicable) The app launches and functions as expected in dev mode (`npm run dev`).
6. [ ] New IPC channels are documented in `spec.md` if they affect the overall architecture.
