# Suggested Commands for NoteVault

## Development

- `npm run dev`: Start the application in development mode with hot-reload.

## Quality & Maintenance

- `npm run typecheck`: Run TypeScript type checking for both main and renderer processes.
- `npm run lint`: Run ESLint to check for code quality and style issues.
- `npm run format`: Automatically format the codebase using Prettier.

## Build & Distribution

- `npm run build`: Build the application for production.
- `npm run build:win`: Create a Windows installer (NSIS) using electron-builder.
- `npm run build:mac`: Create a macOS build.
- `npm run build:linux`: Create a Linux build.
